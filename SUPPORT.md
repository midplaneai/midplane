# Support

Thanks for using Midplane. Here's where to go for help:

- **Questions, ideas, usage help** → [GitHub Discussions](https://github.com/midplaneai/midplane-cloud/discussions)
- **Bugs & feature requests** → [open an issue](https://github.com/midplaneai/midplane-cloud/issues/new/choose) (use the templates)
- **Security vulnerabilities** → **don't** open a public issue; see [SECURITY.md](./SECURITY.md)
- **Commercial / Enterprise (`ee/`) licensing** → sales@midplane.ai

Community support is best-effort. Self-hosting is a single supported artifact
(`MIDPLANE_SELF_HOST=1`); we can't debug arbitrary custom deployments. The
managed cloud is the fully supported path.
